Cashe - папка с файлами для работы с памятью на диске, в частности с кешем, для более быстрой отработки а так же для защиты
Controllers - папка с файлами управления системой движка, соединяет все папки с файлами, создает обмен информацией. Мост среди всех файлов.
Modules - папка с модулями системы, базовой логики движка сайта
REST - папка с файлами для подключения к внешним источникам путем Rest запросов. 
Vendor - папка со сторонними файлами и библиотеками
View - папка с файлами представлений. В ней содержатся файлы с основной разметкой HTML сайтов, со встроенными скриптами.
